# 📊 Modern Admin Dashboard

A responsive, modern admin dashboard built with **React**, **Tailwind CSS**, and **Recharts**.
This project showcases my ability to build clean UI layouts, manage component structure, and visualize data using real-world dashboard patterns.

---

## 🚀 Live Demo

🔗 [View Deployed App](https://dashboard-psi-lyart-34.vercel.app/)
🧠 [View Code on GitHub](https://github.com/NicpaulGacha/Dashboard)

---

## 🧱 Tech Stack

- ⚛️ **React** — component-based front-end framework
- 🎨 **Tailwind CSS** — utility-first styling and responsive design
- 📊 **Recharts** — data visualization library for charts
- 🌐 **React Router DOM** — for routing and page navigation
- ⚙️ **Context API** *(optional)* — for theme toggling (dark/light mode)
- 🚀 **Vercel** — for deployment

---

## ✨ Features

✅ Responsive layout (works on all screen sizes)
✅ Sidebar navigation with routing
✅ Topbar with user profile
✅ Analytics charts (Line, Bar, or Pie)
✅ User data table (sortable, styled)
✅ Clean, scalable component structure
✅ Ready for future integrations (API data, CRUD, auth, etc.)
